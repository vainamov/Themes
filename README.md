# Themes
Themes for the Mozilla Firefox Addon "Stylish".

###Installation

1. Get Stylish https://addons.mozilla.org/de/firefox/addon/stylish/.
2. Go to about:addons > User-Themes > Create new style.
3. Paste the corresponding code.
